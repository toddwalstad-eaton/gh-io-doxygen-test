Detailed Description

An integral mapping between a CDS parameter and one or more Modbus registers.

This class implements a register mapping for time types (on the Modbus side).

Member Data Documentation

bool CTimeMapping::_use_timestamp
private

Normally, a time mapping is used to translate a DATATYPE_TIME value in the CDS into Modbus registers.

However, we can also use it to translate the timestamp of any CDS value; i.e. the internal timestamp of "mValue", rather than the actual value of "mValueTime". If this flag is true, we translate the timestamp rather than the value itself.
