Detailed Description

Modbus PEDS Mapping.

This class encapsulates all of the mapping information specific to a single PEDS file. Each EDS may be utilized multiple times (e.g. in different Modbus-TCP Unit IDs), so the PEDS-specific data is all kept in this structure so that it can be shared between those different instances.
