Detailed Description

SSL Connection class, which manages TCP transactions.

This class handles all TCP transfers between the server and a single client. It is spawned by the listener when an incoming connection attempt is made, and is used to transfer requests and responses to/from the server. Patterned after the CTcpSslConnection class
