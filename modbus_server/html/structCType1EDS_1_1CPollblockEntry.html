Detailed Description

Represents a Data Block (xBlock).

A Data Block consists of one or more Data Block Translation entries. Channels reference Data Block by name and address (such as xBlock[n]). Data Block has dependency to determine its existence with condition. Example of dependency: "if (xModel!=250);".
