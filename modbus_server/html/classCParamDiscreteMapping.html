Detailed Description

A mapping between a CDS parameter and a Modbus discrete/coil.

Each parameter in the CDS can be mapped in various ways into a binary Modbus discrete/coil value, to be located within the exposed Modbus discrete address space. This class implements the translation between the Toolkit's CDS element, and the boolean data used by Modbus.

Translation is bi-directional. Translation from a CDS element into a discrete value is more common, since that occurs for every discrete READ operation. Translation from a binary value into a CDS element occurs for every discrete WRITE operation.
