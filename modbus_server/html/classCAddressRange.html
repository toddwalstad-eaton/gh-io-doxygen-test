Detailed Description

A range of Modbus register / discrete addresses.

This is simply a wrapper on top of a pair of Modbus addresses. It proves useful for doing address matching when we've got overlapping ranges of addresses. For example, if parameter X maps to addresses 1000 through 1009, and parameter Y maps to addresses 1010 through 1019, an incoming request for addresses 1005 through 1012 will need to fetch both X and Y, even though none of the specific register numbers match between those 3 pairs.
