Detailed Description

A mapping between a CDS parameter and Modbus data.

This class is used as a base for register & discrete mappings.

px::string CParamMapping::param_path ( ) const

Getter for parameter path.

Normally the raw path of a param mapping includes the specific attribute to which it maps, for example "mACVAB/Value". Sometimes we only want the attribute name, e.g. "mACVAB", so we use this call instead.
