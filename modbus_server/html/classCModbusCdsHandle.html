Detailed Description

Modbus CDS Handler class.

The comms to the CDS are handled by this class. Every request, both solicited and unsolicited, made to the CDS will eventually trigger a callback to the handle_respnse. The request will then be routed back to be serialized out. Eventually this might be changed out for a ASIO CDS Service
