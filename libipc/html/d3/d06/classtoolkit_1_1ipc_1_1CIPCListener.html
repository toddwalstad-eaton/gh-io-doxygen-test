Detailed Description

Listener for incoming connections.

This class is used to bind to a UNIX socket and listen for attempts to establish new connections. It is used by the server application and will typically exist for the lifetime of the server, accepting and establishing new connections as needed.

void CIPCListener::obtain_lock ( )
private

Obtain a lock on the IPC "lock file".

The IPC "lock file" is just an empty file which is used to prevent multiple copies of the app from running concurrently. Calling this function will open & lock the file.

Exceptions
generic_errThrown if the file can not be opened and/or locked, possibly because another instance of this application is running.
