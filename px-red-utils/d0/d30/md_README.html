px-red-utils Documentation

px-red-utils information

This is the repository to store common libraries and utility files used in various submodules/components of the toolkit.

Structure of this repository

Folder Type Description
libetnlnxutils local Contains linux exec which helps to run linux commands/shell scripts from cpp code
CHANGELOG local A changelog file to track the different versions and importante changes
COPYING local A licence file
Makefile local A top level Makefile for this repository to build
docs local location for various documents we may have
README.md local A readme file for the px-red-utils repository information
buildroot local Contains example of buildroot package configuration for the lib

How to compile outside of the toolkit

This module can be use outside of the PxRed toolkit.

  • make compile libetnlnxutils
  • make install INSTALL_PREFIX=<MY_PREFIX> copy the lib in <MY_PREFIX>/lib and the header in <MY_PREFIX>/include

Configuring your builds with the toolkit

The build options are provided by the following "config" files:

  • px-toolkit/config - contains default Toolkit package choices
    • px-red-utils depends on libipc and libsyslog'
  • px-toolkit/depend - Sets some Toolkit configuration dependencies, based on the configuration choices made.

Build Dependencies on px-red-utils from the PxRed team

  • px-red-certmgr - Device certificate manager repository build depends direclty on px-red-utils.
  • iot_agent - Iot agent module for toolkit depends directly on px-red-utils.
  • sysmond - System Monitoring and Integrity Checking daemon module for toolkit depends directly on px-red-utils.
  • update_linux_image - Update linux image module for toolkit depends directly on px-red-utils.

Output path for the px-red-utils build on the toolkit

  • INSTALL_PREFIX/lib/ - Contains the exported build files for px-red-utils.

Include/Header path for the dependent modules on the toolkit

  • INSTALL_PREFIX/include/ipc - Include path for ipc module dependency.

Library path for the dependent modules ont the toolkit

  • INSTALL_PREFIX/lib/ - IPC build Library path.
  • INSTALLPREFIX/lib/ - SYSLOG build Library path.

Path to look for px-red-utils headers on the toolkit

  • INSTALL_PREFIX/include/etn_lnx_utils/ - Include path for px-red-utils dependent modules to look for.
