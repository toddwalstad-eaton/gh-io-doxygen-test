bool CLinuxUtil::shell_cmd ( const char *  label,
const char *  cmd,
int  timeout,
bool  leave_newline,
int &  cmd_status,
std::vector< std::string > &  outlines 
)
static

Open a shell, run the given command, and return the output and result.

Uses popen() to spawn the shell for the command. The stdout and stderr output will be combined in the output lines returned. Empty lines of output will not be returned.

Parameters
labelA simple text label for debug purposes; eg, the name of the program to be run.
cmdThe command line to be run in the shell
timeoutTime to wait before forcefully terminating the command execution if not already completed If >=0, the given timeout will be considered. Otherwise, no timeout will be implemented.
leave_newlineFlag to indicate whether to retain (if True) or else strip any final newline character from the output lines.
[out]cmd_statusThe integer status value from pclose() which can be further checked with the macros described under wait() in order to determine success/failure of the command executed by popen() (eg, WIFEXITED() and WEXITSTATUS())
[out]outlinesAn array of non-empty line(s) of output from the command.
Returns
A bool to indicate complete success (True if command ran and returned a 0 exit code) If False, the cmd_status can be assessed to determine what failed.

std::vector< std::string > CLinuxUtil::get_rwx_mounts ( const std::string &  mounts_file)
static

Return list of read/write executable mount points.

These are all filesystems that are mounted read/write without "noexec", except for virtual filesystems that root cannot make files in, like sysfs/proc. AUFS or Overlay mountpoints are also excluded (executability is determined by branch FSes).

Parameters
mounts_fileFile to read from (usually "/proc/mounts", parameterized for unit testability)
Returns
List of mount points that meet the criteria, empty if none
Exceptions
Canthrow 'std::runtime_error' exception if mounts_file cannot be opened or cannot be properly parsed
