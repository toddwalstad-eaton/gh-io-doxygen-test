Detailed Description

The overarching namespace for Linux Utilities Library.

All of the exported functionality for Linux Utilities Library is found under this etn:: namespace.

This namespace is further divided by namespaces into the functional areas.

Many of the core toolkit objects, data types, and enums are defined at this namespace level because they are used across this library's functional areas.
