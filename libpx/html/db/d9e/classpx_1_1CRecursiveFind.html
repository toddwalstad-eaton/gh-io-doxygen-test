Define a tuple of file/directory parameters 0th element is filename or childpath (i.e.

the entry->d_name returned from readdir()) e.g. filename of '/tmp/protocol.txt' would be 'protocol.txt' childpath of '/tmp/protocol' would be 'protocol' 1st element is filepath 2nd element is canonical (i.e. real filepath) 3rd element is file size (from stat() st_size value)
