Detailed Description

CDS I/O actor which listens for CDS notifications.

This actor is custom (not based on one of the out-of-the-box Boost.Asio I/O object types). It is designed to signal the main I/O service and cause a handler to be invoked when there are responses ready to be read from the CDS.
