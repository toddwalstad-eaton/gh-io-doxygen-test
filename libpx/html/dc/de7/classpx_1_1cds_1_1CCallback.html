Detailed Description

The abstract Callback prototype for CDS responses.

Must derive callbacks from this class, and register them with the CDS via CClient::register_callback() or CClient::register_unsolicited_callback().

The cds_response_callback() function will be called from a background thread, so use appropriate locking if the callback must access resources owned by your main thread. Keep it quick if possible, so as not to hold up further responses from the CDS.

It will be passed a shared_ptr to the CRequestResponse object, which will include the relevant transaction ID. This will dequeue the response object from the CClient's response_queue.

The CClient destructor will unregister this callback if it's still registered.
