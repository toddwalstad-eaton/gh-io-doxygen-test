Detailed Description

Represents a single subscription (or un-subscripton): the (enumerated) type of subscription, the object it applies to, and which subscriber ID is asking (a CDS client can get more than one subscriber ID if it desires, for multiplexing).

Set "unsubscribe" to true to unsubscribe, the default (false) means to subscribe.

bool px::cds::CSubscribeSingle::is_id_recursive

Indicates that the object id (must be device id) specified also includes its children (device ids) recursively.

This is different than the recursive types in px::sub::subscription_type - setting this to true sets up the subscription on all existing children. Leaving this false but using a recursive subscription type sets up the subscription on all existing children AND on any that are created later.
