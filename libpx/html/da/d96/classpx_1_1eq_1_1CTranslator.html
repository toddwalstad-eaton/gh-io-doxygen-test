Detailed Description

template<typename T>
class px::eq::CTranslator< T >

A translator class which defines the interface between the user's code and the generic equation code.

Note that it is in no way required that you use this class as a base, it is simply convenient in many cases to fill in exception-throwing defaults for the operations that your type doesn't support. The requirement imposed by libtranslator++ is that your type conforms to the "Callable" concept (which this class does) and that the template value_type conform to "Translatable".
