Detailed Description

This class define a container for Device data from the CDS.

It goes beyond the px::CDevice to include state and channel information. It can also be checked during device subscription updates to see if it (or its children) have been changed and must be refreshed.

This container will be used in the CDS Device Map provided and maintained by CCdsHandler.
