Detailed Description

Example implementation of "Translatable" object using CSimpleVal as a double.

The genetics of this class are:

  • CSimpleVal (with double template type) supports the operations on the value
  • CTranslator layers the get/set base operations, using templating, onto CSimpleVal
  • This class inherits all that and provides a very simple example implementation that "handles" real values.

You will want to create a handler class like this that implements get() and set() and attaches real values to them somehow.
