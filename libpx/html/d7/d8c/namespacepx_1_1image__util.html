std::string &  error 
)

Check an image file using data from the provided image.sums.

Tests size and whichever of sha1sum or sha256sum is present, or both.

Parameters
image_pathPath to image file to check. Must not be empty.
image_sumsString containing contents of the image.sums file Must not be empty.
image_typeThe Type of this image as listed in image.sums e.g. "rootfs" Must not be empty.
[out]errorIf false is returned, this is set to an informative message
Returns
True if the image passes all checks, False if any checks fail, the entry is not found in image_sums or has no checksums there, or there is an error.
