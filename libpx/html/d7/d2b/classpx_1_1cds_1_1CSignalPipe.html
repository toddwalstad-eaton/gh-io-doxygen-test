Detailed Description

Class for catching UNIX signals & delivering them to the main polling loop.

This utility class wraps the functionality provided by C-style signals in a class that provides event readiness notification when a signal has been received (via select(), epoll(), etc.) This simplifies signal handling by allowing the main event loop to be awaken when a signal is received.
