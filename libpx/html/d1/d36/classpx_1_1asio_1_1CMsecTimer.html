Detailed Description

Generic application timer actor used to periodically call back to the application code (which has the CTimedObject implementation).

This class creates a free-running timer with the given msec period and calls the CTimedObject timeout() function whenever the timer expires, then restarts the timer.

void px::asio::CMsecTimer::timeout ( const CIoTimer &  timer)
virtual

Callback invoked when our internal timer times out, at which time we callback the application's timeout function.

Ie, this is the first in a two-step chain of timeout() functions; the application also implements this same function.

We will reset the timer as long as it is non-zero and run again.

void px::asio::CMsecTimer::update_timeout ( unsigned int  msec_timeout)

Cancel any pending timeout and restart the timer with the given timeout period.

Setting the timeout to 0 will stop the timer immediately. If this is called for a stopped timer, it will be restarted.
