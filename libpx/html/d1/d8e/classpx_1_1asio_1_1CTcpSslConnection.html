Detailed Description

TCP Connection class, which manages TCP transactions.

This class handles all TCP transfers between the server and a single client. It is spawned by the listener when an incoming connection attempt is made, and is used to transfer requests and responses to/from the server. Patterned after the CTcpConnection class

Write the data from the outbox to the service.

This operation is implemented in terms of zero or more calls to the stream's async_write_some function, and is known as a composed operation. The program must ensure that the stream performs no other write operations (such as async_write, the stream's async_write_some function, or any other composed operations that perform writes) until this operation completes.
