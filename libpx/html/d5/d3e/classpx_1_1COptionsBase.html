Detailed Description

Configuration options base class.

This class handles parsing & storage of configuration options, typically set via the command-line.

staticprotected

Breaks a line into multiple components at convenient break points.

This call will break a string into multiple sub-strings, each one capped at a maximum length. It tries to do so intelligently, so that breaks occur at spaces rather than mid-word. This is used to produce nicely-formatted left-justified text blocks when showing command-line usage. Linefeeds ("\n") included in str will always cause a break at that location.

Parameters
strThe string to parse, generally all a single line. Must not be empty.
ncharsThe maximum width that each resulting line can be
sepA list of separator characters at which we can conveniently break a line into pieces. Must not be empty.
Returns
A vector of sub-strings, each limited in length
