Detailed Description

Monotonic time.

This is a particular kind of timestamp, known to be derived from the monotonic clock. It is functionally the same as a CTimestamp object for the most part, but making it its own type serves both to make it clear in code what kind of timestamp we're dealing with, as well as prevent accidental comparisons between monotonic and wall-clock times.
