virtual void CSerialConnection::read_more ( boost::asio::mutable_buffer &  buf)
finalvirtual

Read more data from the serial input channel.

This call initiates an asynchronous read operation, filling the given receive buffer with any additional data that can be read. This is the final implementation of the virtual function invoked by the active I/O message (_queue.front()).

The I/O message provides us with the buffer into which data should be read; the messenger itself does not manage the buffer. This allows each message to be self-contained.

Parameters
bufThe buffer (whose allocated memory actually resides in the originating CIoMessage) which will hold the response as it is read in.

Write data to the serial output channel.
