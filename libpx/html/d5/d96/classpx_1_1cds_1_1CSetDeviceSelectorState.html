Detailed Description

Set state for all devices covered by a DeviceSelector object.

Note
The DeviceSelector has a flag for recursive, but setting DeviceState is inherently recursive, so it is redundant. With one caveat: if a specified device is already at the specified state, nothing will be done and its child devices will not be affected.
