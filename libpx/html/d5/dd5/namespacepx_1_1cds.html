Detailed Description

px::cds is the namespace for all the classes that provide the object types and client and server functions for Power Xpert Toolkit applications to communicate with its Core Data Store (cds).

px::cds contains the CDS API objects - these are passed to and from the CDS using px::cds::CClient::submit() and related functions, to request that the Core Data Store do stuff.

Typedef Documentation

using px::cds::response_callback_t = typedef std::function<void ( std::shared_ptr<CRequestResponse> )>

The typedef of the standard CDS response callback function.

The callback function has one argument, a shared pointer to a (derivative of) the CRequestResponse class, containing the response information. The callback has no return (void).
