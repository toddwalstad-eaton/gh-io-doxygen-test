px::string px::CProtoOPS::GetFields ( size_t  remove_layers,
const googlePB::Message &  msg 
)
static

GetFields returns the internal text of a protobuf message, stripping away as many outer layers as requested, and removing quote marks.

This function will start with the protobuf DebugString() output and remove outer layers of protobuf metadata. Eg, to show just the fields of an OOB protobuf, set remove_layers to 3, so ACCESS_CTRL/HTTP of { "toolkitStruct":{ "access_ctrl":{ "restricted":false, "port":80, "trusted":[ "10.222.41.58"] , "label":"HTTP" } } } becomes just restricted:false, port:80, trusted:[ 10.222.41.58] , label:HTTP

Parameters
remove_layersThe number of outer layers (ie, braces) to remove.
msgProtobuf object to be processed (dumped)
Returns
String containing the inner text of the requested protobuf
