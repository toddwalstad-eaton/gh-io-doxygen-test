Detailed Description

Delete trend storage by date range and channel ID.

Trend storage is automatically deleted for channels that are deleted, and to clean up space if trends are old or the flash is getting full (per settings in OOB tag DatabaseSettings). This request can be used to clean up trends for existing channels where the data is no longer needed, to free up storage space.

There is no response other than overall RequestStatus.

Trend data for the current and previous day is not deleted, even if otherwise covered by the parameters of the deletion request.
