Getter for this canonical string associated with this enum.

Enum to string conversions are supported via an inverse lookup table that maps to the same string entries used by the parse() function. The lookup table is built as a static member, i.e. initialized the first time any part of your program calls this function. After that, string conversions are extremely fast.

void px::COverrideLevel::parse ( const char *  str,
unsigned int  len 
)
private

Parse an input unit string and assign the corresponding enum.

This function utilizes a gperf-generated perfect hash table to perform an efficient lookup of the given canonical unit string, and find its associated enumeration value. Comparisons are case- sensitive, and must match the corresponding canonical string exactly or they will be considered invalid. The result is assigned to _val.

Parameters
strThe canonical string name of the assigned enum. Must not be null nor empty.
lenLength of the string; if 0, length will be determined automatically by looking for a NUL terminator
Exceptions
generic_errThe provided string is unknown or invalid
