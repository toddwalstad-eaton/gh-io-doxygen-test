Detailed Description

Iterator into a sorted string table or key/value database.

This class is a simple wrapper for the underlying implementation's iterator functionality. It's primarily useful because we can use it to enforce good programming, by detecting mis-use when the original table or database is closed while there are still outstanding iterators present.

void px::db::CIterator::close ( )

Close the iterator.

This is often not needed, since the iterator will be closed upon destruction. However, it's useful both for unit testing, and in scenarios in which (for whatever reason) you have an iterator object which must live beyond the lifetime of the table/database that it was acquired from.

This call will trigger an assertion failure if the iterator points to a table/database that is no longer open.
