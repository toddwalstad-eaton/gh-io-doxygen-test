Detailed Description

template<typename Translator, typename TransVal, bool TestEval>
class px::eq::detail::eval< Translator, TransVal, TestEval >

This visitor class is used to evaluate an expression and produce some result.

The result type is templated, and must implement all of the operations expected here.
