Detailed Description

Delete a list of alarms.

Client either fills in a list of closed alarm ids to be deleted or specifies the closing time, level and number of alarms to retain. Alarms satisfying the given criteria are deleted from the database. There is no response other than the "status" member of the returned message.

Delete Alarm by ids deletes all the sequesnces of a particular alarm.
