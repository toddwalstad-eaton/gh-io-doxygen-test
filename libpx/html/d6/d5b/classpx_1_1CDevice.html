Detailed Description

Represents a device, which can be either a publisher, a device connected to the publisher, or a network connector such as INCOM port 1.

Devices contain channels and possibly other devices. Every system has at least one device, the "publisher" at ID 1, which has no parent.

uint64_t px::CDevice::change_signature

Represents a change signature, to help avoid any unnecessary re-discovery by indicating if any significant change has occurred in this device branch.

This field shall not change if the only changes are the device's family, model, physical address, serial, vendor, or asset_id.

Changes to fields that trigger a change to change_signature will change this device's parent's change signature, and so on up to the root of the tree.
