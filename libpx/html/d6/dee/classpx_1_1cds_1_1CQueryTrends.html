unsigned int px::cds::CQueryTrends::interval

Return data for the specified interval, in minutes.

Valid values are 5, 15, 30, 60, 120, 240, 480, 720, or 1440 minutes, per CTrendStore::QueryTrends. If data is available at a smaller interval, it will be aggregated into the requested interval. If data is only available at a larger interval, that larger interval will be returned.
