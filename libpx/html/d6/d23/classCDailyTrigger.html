bool CDailyTrigger::in_range ( const px::CWallTime &  start,
const px::CWallTime &  end 
)
virtual

Checks whether the "end" time (presumably "now") is within the next triggering window since the "start" time.

As an example, if this trigger was configured for hour_24 = 8 and minute = 33, given a start time of 08:31 (hours and minutes; we can ignore seconds), if "end" time is 08:32, will return False since this is not yet time to trigger for the next interval. However, an "end" time of 08:34 would return True since it is at or past 08:33.

Parameters
startThe starting time (UTC), presumably when last checked.
endThe ending time (UTC), presumably now, to be checked.
Returns
True if end is at a point where it falls into the "next" trigger interval after the "start" time, or it is precisely on that trigger point.
