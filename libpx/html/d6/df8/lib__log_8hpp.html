enum libpx_log_lvl_t

These values are used to define lib "api's" whose verbosity can be varied individually, to get more specific information from the server without overloading the user with output.

Note: when updating this array, be sure to update the corresponding array COptions::subsystem_strings. – TB: How does this apply to LibPX?
