Detailed Description

template<typename T>
class CSharedHandle< T >

This class is used to pass a handle to an object through a thread and back again, then safely get at the original when needed.

It is not thread-safe; This is only meant for objects which are used in the main thread then re-used at some later time when it's uncertain whether the object originally exists. Worker threads should not use the handle to access the original object.
