typedef std::pair<boost::tribool, iobuf_const_iter_t> px::asio::parse_result_t

Pair type used to represent the result of a parse() operation.

The tribool value indicates the success or failure of the parse, while the iterator indicates the point at which parsing stopped/failed.

The tribool value is true if the input buffer is complete and valid; false if the input buffer is invalid; indeterminate if the input is incomplete (and the iterator should point to the start still).
