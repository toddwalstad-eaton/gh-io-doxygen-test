Detailed Description

Since equation evaluation is template-based, the value is free to be any type which implements the required interfaces.

This class serves as a base for such types.

Note that it is in no way required that you use this class as a base, it is simply convenient in many cases to fill in exception-throwing defaults for the operations that your type doesn't support. The requirement imposed by libpx is that your type conforms to the "Translatable" concept (which this class does).
