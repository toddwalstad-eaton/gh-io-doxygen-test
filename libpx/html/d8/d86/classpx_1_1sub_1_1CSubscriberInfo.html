Detailed Description

Per-subscriber information.

This class tracks information for each subscriber, including a user-visible subscription ID and a list of all current subscribed-to items. The latter allows the subscription subsystem to clean up data structures when a subscriber is removed; without this info, a badly-behaving client (for example) would cause a build-up of extraneous information in the internal subscription data structures.
