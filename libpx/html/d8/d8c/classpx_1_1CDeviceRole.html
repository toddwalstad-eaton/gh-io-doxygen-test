Detailed Description

Device role.

See px::device_role for the enum values used.

This class enumerates all of the device roles available within the Toolkit (and exposed via Mercury). This list of roles may be expanded in the future as needed.

Objects of this type are intended for use in lieu of the raw enumeration values wherever possible. Despite being first-class objects, they occupy only 1 byte, and impose virtually no overhead.

const char * px::CDeviceRole::str ( ) const

Getter for this canonical string associated with this device role.

Enum to string conversions are supported via an inverse lookup table that maps to the same string entries used by the parse() function. The lookup table is built as a static member, i.e. initialized the first time any part of your program calls this function. After that, string conversions are extremely fast.

void px::CDeviceRole::parse ( const char *  str,
unsigned int  len 
)
private

Parse an input device role string and assign the corresponding device role enum.

This function utilizes a gperf-generated perfect hash table to perform an efficient lookup of the given canonical device role string, and find its associated enumeration value. Comparisons are case-sensitive, and must match the corresponding canonical string exactly or they will be considered invalid. The result is assigned to _val.

Parameters
strThe canonical string name of the assigned device role. Must not be null nor empty.
lenLength of the string; if 0, length will be determined automatically by looking for a NUL terminator
Exceptions
generic_errThe provided string is unknown or invalid
