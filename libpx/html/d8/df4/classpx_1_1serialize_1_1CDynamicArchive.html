Detailed Description

Dynamic, self-sizing data archive.

This archive performs its own memory allocation, as needed, to store serialized objects. It is easy to use and doesn't require prior knowledge of the size of the objects to be serialized; however, it may re-allocate memory repeatedly, and is therefore not as efficient as the "raw" archive.
