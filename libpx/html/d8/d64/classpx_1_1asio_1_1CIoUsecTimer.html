Detailed Description

The I/O Microsecond Timer class for ASIO Actors.

This class implements an async I/O microsecond timer that is bound to a specific object. This is typically used to place an upper bound on some I/O operation.

void px::asio::CIoUsecTimer::cancel ( )

Cancel the timer if currently running.

The timer can be restarted after this by calling reset() with some new timer delay.

The handle_timeout() will normally be called when this is done, but will recognize that it was canceled and not call its timeout function.

void px::asio::CIoUsecTimer::wait ( )
private

Perform an asynchronous wait on this timer.

This must be called in order for the timer to be useful. The only reason it's not invoked automatically by the constructor is that shared_from_this() doesn't work yet at that point. Using the factory function hides this detail.

This function returns immediately; when the timer expires, handle_timeout() will be invoked.
