Detailed Description

Watchdog timer actor used to periodically ping the CDS.

By periodically pinging the CDS, we assert that this application is still "alive". Otherwise, if enabled, the CDS watchdog function will be triggered and result in a system reboot.

We register with the CDS Watchdog using the given "timeout" value, but, within this process, we set a timer at 1/3 of "timeout" and use that timer to trigger our pings to the CDS Watchdog. Using 1/3 of the full timeout gives us margin and 2-3 chances to get our message through to the CDS.

void px::asio::CWatchdogTimer::close ( )
virtual

On a graceful close, unregister this app's watchdog from the CDS so we won't get an unintended watchdog reboot.

This will gracefully close the actor. It is called automatically by the actor's destructor, but any resulting error information will be lost in that case.
