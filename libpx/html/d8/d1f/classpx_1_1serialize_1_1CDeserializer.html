Detailed Description

template<class Archive>
class px::serialize::CDeserializer< Archive >

De-Serializer object.

This is a thin wrapper on top of the archive class. It simply redirects the & operator to the load() function in the archive class, in order to de-serialize each object's data from an input stream.
