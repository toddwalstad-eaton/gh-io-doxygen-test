Detailed Description

Real-time scheduling policy class.

This class encapsulates settings for real-time scheduling, including the class of scheduler to be used and the priority level.

Constructor.

This constructor parses the user-supplied policy and priority strings and stores them in canonical internal form. Priorities here range from -20 to 19, where a lower value is higher priority, and the default level is 0.

Parameters
policyReal-time scheduling policy.
priorityReal-time scheduling priority level (numeric).
niceNice-level for normal scheduling classes (numeric). See the manpages for 'nice' for further explanation.
Exceptions
generic_errThe supplied policy and/or priority strings are invalid (usage error).
