Enumeration Type Documentation

enum px::dep::update_type : uint8_t
strong

The dependence "type" between a source and target node in the graph.

Real-time dependencies are the usual case, in which a target node's value depends on a source node's latest value.

Interval dependencies occur when a target node's value depends on a source node's interval trait; for example, we might calculate a 5-minute energy reading by using the previous-interval average of a power channel.
