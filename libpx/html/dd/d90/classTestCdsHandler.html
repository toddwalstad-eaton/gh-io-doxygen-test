virtual void TestCdsHandler::handle_response ( std::shared_ptr< px::cds::CRequestResponse >  resp)
virtual

Called by the ASIO io_service when one or more (normally expected) responses or subscription updates are sent from the CDS.

This will be invoked for the subscription events, or responses to submit(), or (in unusual cases) an unsolicited response from the CDS itself.

Once you have started the ASIO service, this function as well as internal_handle_response() will be given any subscription updates, so your implementation must validate that this message was for one of your own subscriptions.

Your derived class must implement this function and create a CCdsConnectionCallback() instance to get this callback to work.

Parameters
respThe response from the CDS (wrapped in a shared pointer). Must not be null. The resp type is aliased as cds::response_ptr_t

Implements px::asio::CCdsHandler.
