Detailed Description

Singular get/set field request.

The API object will contain a list of these.

CSingleField Use Examples:

Device/Channel: data.type = device or channel, data.field = "canonical_model_name", data.id = 2 - Device or Channel ID, data.tag = N/A

Channel Value By Name, using any Device ID: data.type = channel_name, data.field = "value", data.id = 2 - Device ID, data.tag = "mPhaseAngleVab"

Channel Value By Name and same Device ID for all channels: data.type = channel_value_by_name_group_id, data.field = N/A, data.id = 2 - Device ID, data.tag = "mPhaseAngleVab",

Channel Value By ID: data.type = channel_value_by_id, data.field = N/A, data.id = 122 - Channel ID, data.tag = N/A

Command: data.type = command, data.field = "cSARstEnergy", data.id = cmd handle,

Config: If Request tag is 'SERIAL_PORT/2', then: data.type = config, data.field = "protocol", data.id = 2, data.tag = "SERIAL_PORT"

If Request tag is 'NETWORK', then: data.type = config, data.field = "use_dhcp_host_name", data.id = FIELD_ID_NOT_USED, data.tag = "NETWORK"
