Detailed Description

Represents a light weight accumulator to calculate min, min_time, max, max_time, avg, and last values for CValue.

Typically, the values are accumulated for a fixed period of time (such as 5 minutes).

Get the count, avg, min, min time offet, max, max time offset, and last values from this interval.

The time offset is 1/10 seconds to the interval timestamp. If the count is zero, the values will all be NAN.

Parameters
[out]interval_timestampThe (ending) time, in UTC minutes, for the present accumulator.
[out]countThe count of samples in the interval
[out]avgThe accumulated average value, if any (else NAN).
[out]minThe minimum value for the interval (so far), if any (else NAN).
[out]min_time_offsetThe time offset (remaining) until the interval timestamp, for the minimum reading, in 1/10 seconds.
[out]maxThe maximum value for the interval (so far), if any (else NAN).
[out]max_time_offsetThe time offset (remaining) until the interval timestamp, for the maximum reading, in 1/10 seconds.
[out]lastThe last (aka "actual") update value for the interval, if any (else NAN).

The time offset is 1/10 seconds to the interval timestamp.
