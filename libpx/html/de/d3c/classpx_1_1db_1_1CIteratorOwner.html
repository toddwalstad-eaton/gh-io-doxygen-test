Detailed Description

Per-database iterator "owner" class.

This stub class provides no functionality, but it's very useful to have a common "owner" for iterator objects. That way we can use the semantics of weak_ptr to track the validity of outstanding iterators into a database, ensuring that they're not abused in calling code.
