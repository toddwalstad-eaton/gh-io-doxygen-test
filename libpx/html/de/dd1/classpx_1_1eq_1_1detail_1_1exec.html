Detailed Description

template<typename Translator, bool TestEval>
class px::eq::detail::exec< Translator, TestEval >

This visitor class is used to execute a statement.

Statements are analogous to those in C programs: they perform some function, and (potentially) have side effects. Upon completion of one statement, control is returned to whatever follows.
