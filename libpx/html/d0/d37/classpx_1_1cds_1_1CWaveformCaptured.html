Detailed Description

Used by device handlers to notify CDS of a waveform-capture event.

When called, a one-sequence alarm event will be posted, already closed, on the device's "sWaveformAvailable" channel. The channel's value is set to the provided waveform_uri. The waveform_uri is also included in the posted px::CAlarmUpdate.

Populating the waveform URI, and retrieving it, are done outside of the CDS.
