px::CRandom::CRandom ( const boost::optional< unsigned int > &  seed = boost::none)

Constructor.

Parameters
seedThe PRNG is initialized using this seed value. If none is provided, an internal mechanism will be used to generate a truly random seed (via /dev/random or comparable). You can find out this original seed (for use in a subsequent run) at any time via get_seed().
Exceptions
generic_errCouldn't seed PRNG from /dev/random
