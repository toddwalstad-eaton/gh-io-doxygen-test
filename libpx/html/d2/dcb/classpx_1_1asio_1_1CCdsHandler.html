Detailed Description

ABC for CDS Handler, provides a common synchronous/asynchronous interface to the CDS API.

Synchronous CDS utility functions are imported from the base CCdsSyncIO class.

– Note: Using this class requires ASIO services –

Every ASIO based application should derive a handler from this, and must implement the virtual function for handle_response(). Provides the basic functions for handling OOBs and subscriptions, and for discovering and maintaining the map of Device and Channel data.

EXCEPT if you want to use the submit() and its handle_response() callback. The latter will be will be invoked via the associated px::asio::CCdsConnectionCallback, created by the px::asio::CCdsConnection, which uses ASIO functionality.
