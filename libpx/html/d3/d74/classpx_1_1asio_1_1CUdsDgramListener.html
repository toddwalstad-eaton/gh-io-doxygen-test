Detailed Description

Unix Domain Socket (datagram) Listener class

This class implements a UDS "listener".

It listens to a specific socket, of type datagram_protocol::socket, and manages responses to the incoming requests. In the normal case, we receive a request on this port, process it, and send a response (unicast) back to the sender.

Note: has no current uses; may need work. However, it has some useful concepts, particularly in the storing of the original address of a received packet. Perhaps it can also be merged with CUdpListener, which seems to have the same intent.
