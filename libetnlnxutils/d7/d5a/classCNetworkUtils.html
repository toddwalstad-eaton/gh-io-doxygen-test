bool CNetworkUtils::string_to_mac ( const char *  mac_text,
uint8_t *  mac_buffer,
uint16_t  buflen 
)
static

This function extracts the MAC address bytes from the received 'mac_text' buffer (xx:xx:xx:xx:xx:xx) and put into the 'mac_buffer' buffer provided by the user.

Static function to convert text for a MAC address to an array of bytes.

(eg "0A:DE:30:83:BA:C0")

Parameters
mac_text[IN]- The input buffer holding the MAC address bytes with colon-separated string of hex digits
mac_buffer[OUT]- Output buffer to hold MAC address bytes
buflen[IN]- input buflen is the buffer length of mac_buffer
Returns
True - if the MAC conversion is successful, False - Otherwise

eg "0A:DE:30:83:BA:C0"
