Detailed Description

A class to store the structures for one Channel's Real-Time and Trend messages, along with the storage for the character string data pointed to by these IoT structures.

Also implements the callback handlers for channel settings updates.

const void * CIotChannel::get_iot_channel_item_for_realtime_value ( bool  for_trend = false)

Gets a pointer to either the IOT_DATA_CHANNEL_REALTIMES_ITEM member of this instance, or else the IOT_DATA_TREND_ITEM member with its "actual" value, after it has been unconditionally updated with the latest channel present value and timestamp value.

Note that in the for_trends case, the real-time channel value is still what is used, not a trend value from the CDS, and that only the "actual" field is used (min/avg/max are nulled out).

Parameters
for_trendIf for_trend is true, will return an IOT_DATA_TREND_ITEM structure
Returns
Pointer to the internal IOT_DATA_CHANNEL_REALTIMES_ITEM or IOT_DATA_TREND_ITEM structure. Will return null pointer if called before we have initialized the CDS data reference for this channel.
