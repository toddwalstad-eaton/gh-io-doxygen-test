1 # Interface to the Eaton IoT Device SDK {#md_iot_sdk}
2 
3 The full Eaton IoT Device SDK is located as a git submodule in the docs/platform_device_sdk folder.
4 
5 The "make" process starts by copying only the 13 files of interest out of the SDK, from the locations shown under iot_device/include and source, and puts them in the main project's include/sdk and source/sdk folders; they are compiled from these locations and linked to iot_agent.
