11 # 2. Overview
12 
13 ## Data server to manage Internet-of-Things activity (IoT Agent)
14 
15 This agent, if it has a valid Azure IoT Hub Connection String, will initiate the
16 connection to that IoT Hub via an AMQP (or MQTT) WebSocket connection.
17 Once the connection is made, the Agent will publish messages to the Azure cloud,
18 where they will be received by PX White services; these are Device-to-Cloud (D2C) messages.
19 PX White can also send messages from the cloud to the device (C2D), and these will be received
20 by the Agent and responded to.
21 
22 The IoT Agent also includes "Net Diagnostics" code to try to determine the cause of any
23 failure to connect initially or of any disconnect experienced while connected.
