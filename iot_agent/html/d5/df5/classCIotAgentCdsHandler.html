Detailed Description

IoT Agent's CDS Handler class (an ASIO Actor, specialized for iot_agent).

The communication with the CDS is handled by this class. Many of these methods use synchronous CDS calls (submit_wait()) for the programming flow convenience.

Any asynchronous callback, both solicited and unsolicited, from the CDS will be managed by the handle_response() method.
