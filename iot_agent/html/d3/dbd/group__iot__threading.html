enum px::proto::IotBlobUploadErrorEnums BlobUploadManager::launch_blob_upload_thread ( const px::string &  upload_oob_tag,
const px::proto::IotBlobUpload &  blob_upld_req 
)

Launches the thread that does the blob upload operations.

This will be a run-to-completion task in a new thread, which will allow the iot_agent main thread to continue to run and publish regular data while the blob is uploaded on a slow background thread. Eventually this thread returns a result code and a useful message, via get_results().

Parameters
upload_oob_tagIotBlobUpload file specific tag
blob_upld_reqIotBlobUpload OOB structure pertaining to a particular file being uploaded
Returns
The result of the launch; will normally be INITIALIZED, unless the thread launch fails, in which case it will be DONE_ERROR, and probably something is terribly wrong in the system.

enum CIotDeviceClient::client_states CNetDiag::launch_net_diagnostics ( )

Launches the task that runs the net diagnostics tests.

This will be a run-to-completion task in a separate thread, which will allow the iot_agent main thread to continue to run and accept signals while the lengthy diagnostics tests are carried out. Eventually this thread returns an error code and a useful message, via get_results().

Returns
The client state to use now; will normally be IN_NET_DIAGS, unless the thread launch fails, in which case it will be DO_EXIT, since something is terribly wrong in the system.
