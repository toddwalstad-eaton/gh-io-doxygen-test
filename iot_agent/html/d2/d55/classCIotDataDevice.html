Detailed Description

Class to provide the IOT_DATA_DEVICE_ITEM instance from the CDS data for one Device.

Manages its memory (holding the original data until destroyed). Has methods to access the real-time and trend data for all channels owned by this device.

enum CIotDataDevice::idd_state_t

Define the online/offline "IoT Agent" states for a CDS device.

These states are used to capture the transition of a CDS device from a communicating to non-communicating state, and allows us to send a last known channel value update to the cloud.

void CIotDataDevice::clear_iot_subdevices ( )

Clear subdevices structure, recursively.

Since we don't allocate the structure for each subdevice (instead, we point to its CIotDataDevice::_iot_device_struct), which are known for Azure as the "list items", we don't have to free the list items. HOWEVER, the list itself allocates a LIST_ITEM_INSTANCE structure for each entry in the subDevices list (these are IOT_DATA_DEVICE_ITEMs), and this list structure must be released. BUT FIRST we must walk the subDevices list and do the same for any subDevices list of its entries, all the way down.
