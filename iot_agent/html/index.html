2. Overview

Data server to manage Internet-of-Things activity (IoT Agent)

This agent, if it has a valid Azure IoT Hub Connection String, will initiate the connection to that IoT Hub via an AMQP (or MQTT) WebSocket connection. Once the connection is made, the Agent will publish messages to the Azure cloud, where they will be received by PX White services; these are Device-to-Cloud (D2C) messages. PX White can also send messages from the cloud to the device (C2D), and these will be received by the Agent and responded to.

The IoT Agent also includes "Net Diagnostics" code to try to determine the cause of any failure to connect initially or of any disconnect experienced while connected.
