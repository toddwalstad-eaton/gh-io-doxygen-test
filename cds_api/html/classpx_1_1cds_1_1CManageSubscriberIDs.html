Detailed Description

This message is used to add and remove subscription IDs.

Fills in a number of new IDs to request (this may be zero), and a list of IDs to remove (removing an ID will cancel all its active subscriptions).

The CDS fills in a list of new IDs, if any were requested, and a list of all currently-allocated subscription IDs for this client. Any uses of px::cds::CSubscribe need a subscription ID, a client application may use one or many IDs. (For example, an application managing multiple network connections may wish to use separate IDs to distinguish which network connection needs to be notified about a particular event.)
