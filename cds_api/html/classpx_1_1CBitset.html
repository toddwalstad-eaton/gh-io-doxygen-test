Detailed Description

template<class T>
class px::CBitset< T >

Stores a bit-field of flags defined by an enumerated type.

It is used to manage multiple bit flags efficiently, as a slightly more elegant alternative to bit-fields. We hand-roll a quick & dirty implementation below rather than using std::bitset, because it allows more efficient serialization (std::bitset doesn't provide access to raw storage, so we're forced to use 1 byte per flag).

The template parameter must define "size" to represent the total number of flags present. This is normally done by just placing it at the end of the list, for example:

enum class flag : uint8_t { flag1=0, flag2, flag3, size };
