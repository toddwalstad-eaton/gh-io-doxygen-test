Detailed Description

Sent, unsolicited, from CDS to clients in response to a subscribed event or change, or a login session ending.

Contains information on the change that happened, and the subscriber ID used to subscribe to that change. A client may receive multiples of this message for the same event, if different subscriber IDs were used to subscribe to it.
