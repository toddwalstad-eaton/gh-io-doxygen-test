Detailed Description

Remove a user from the system.

Client fills in a list of user IDs to delete. CDS fills in a list of user IDs and success/failure indication.

Deleting a user will fail if that would leave the system without anyone with security_admin permission.

Permissions Required:
px::permissions::security_admin
