Member Data Documentation

uint32_t px::CChannelTrend::time

Time of the end of the trend interval.

Minutes since 1970-01-01 00:00 UTC.

uint32_t px::CChannelTrend::min_time_offset

The offset (in 1/10 seconds) from the beginning of the trend interval to the timestamp of the min value.

Set to zero if the offset is unknown, or happens to be zero. 32bit for the daily aggregated min/max's time offset, 16 bit for all others.

uint32_t px::CChannelTrend::max_time_offset

The offset (in 1/10 seconds) from the beginning of the trend interval to the timestamp of the max value.

Set to zero if the offset is unknown, or happens to be zero. 32bit for the daily aggregated min/max's time offset, 16 bit for all others.

uint16_t px::CChannelTrend::period

The length of this trend interval, in minutes.

Should be 0 (for unknown), 5, 15, 60, or 1440.
