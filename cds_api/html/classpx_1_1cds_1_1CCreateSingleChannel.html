bool px::cds::CCreateSingleChannel::in_mcl

Do we expect to find a channel of this name in MCL? If true, the channel settings will be filled in with MCL data, and channel creation will fail if that name is not in the MCL.

If false, the channel will be created with blank settings, and will need to be updated later using px::cds::CUpdateChannelSettings.
