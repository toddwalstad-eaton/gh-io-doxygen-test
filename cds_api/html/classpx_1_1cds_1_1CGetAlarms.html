Detailed Description

Returns alarms updates in the system (active or both active and historical) according to the ANDing of all criteria specified.

Permissions Required:
px::permissions::data_read

uint64_t px::cds::CGetAlarms::start_global_count

Return only updates whose global count is greater than or equal to this.

(if go_backwards and only_latest_update are false) or less than or equal to this (otherwise). Default = 0 (no global count filter).

CWallTime px::cds::CGetAlarms::start_time

Return only updates whose timestamp is greater than or equal to this.

Default (left default-constructed) = start at the beginning of time

CWallTime px::cds::CGetAlarms::end_time

Return only updates whose timestamp is less than or equal to this.

Default (left default-constructed) = end at the end of time

uint32_t px::cds::CGetAlarms::max_result_size

Return at most this many updates.

Return oldest (smallest global counts) N if go_backwards=false, newest N if go_backwards=true or only_latest_update=true Default = 0 = no limit (in a long-running system, this may result in px::cds::RequestStatus::RESPONSE_TOO_BIG)

std::vector<px::id_t> px::cds::CGetAlarms::ids

Include updates only from these devices and channels.

Default = empty = all devices/channels

CWallTime px::cds::CGetAlarms::start_occurred_time

Include updates only from alarms whose start_occur_time is greater than or equal to this.

Default (left default-constructed) = unrestricted

CWallTime px::cds::CGetAlarms::end_occurred_time

Include updates only from alarms whose start_occur_time is less than or equal to this.

Default (left default-constructed) = unrestricted

bool px::cds::CGetAlarms::only_latest_update

Include only one update per alarm ID - the most recent.

Default = false If true, we also act like go_backwards is true - start_global_count is the largest GC that will be returned, not the smallest, and updates are returned newest to oldest.

bool px::cds::CGetAlarms::only_first_update

Include only one update per alarm ID - sequence index 0.

If only_latest_update is also set, only include updates for alarms that have only one update. Default = false

bool px::cds::CGetAlarms::go_backwards

If true and we hit the max_result_size, return the go_backwards updates that match the criteria, else the earliest.

When go_backwards is true, start_global_count is still considered the starting global_count, but for traversing backward in time. This flag also affects the order of the alarms returned in the vector: if go_backwards is true, the alarms returned will be newest first, otherwise oldest first. Default = false

uint32_t px::cds::CGetAlarms::starting_sequence_index

Include only updates whose sequence index is greater than or equal to this.

Default = 0

bool px::cds::CGetAlarms::populate_str_val

If true, populate the str_val field for each of the alarms.

Default = false

size_t px::cds::CGetAlarms::count

Number of alarms matching this query request.

Count can exceed max_result_size if count_all is specified.

bool px::cds::CGetAlarms::count_all

If true, when calculating the count that is to be returned, ignore max_result_size, i.e.

keep counting even after we've stopped collecting alarms to return. Default = false

uint32_t px::cds::CGetAlarms::trigger_level

If not 0, filter to only returns alarms equal to this CAlarmTrigger level.

See gt_equal. Default = 0

px::CAlarmTriggerType px::cds::CGetAlarms::trigger_type

If not invalid, filter to only return alarms of this CAlarmTrigger trigger_type.

Default = px::alarm_trigger_type.invalid

bool px::cds::CGetAlarms::gt_equal

If true, return alarm updates greater or than equal to the specified trigger_level, otherwise only equal.

Not applicable when trigger isn't specified. Default = false
