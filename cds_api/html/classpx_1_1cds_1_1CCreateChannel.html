Detailed Description

Create one or more channels.

Fill in a list of CCreateSingleChannel objects describing the channels to be created. There can only be one channel with a particular canonical_name per device.

CDS fills in a list of CChannelSettings objects, including ID, for each of the (possibly zero) successfully created channels.

Permissions Required:
px::permissions::install or px::permissions::internal_app

std::vector<px::CChannel> px::cds::CCreateChannel::created_channels

Channels successfully created, if any.

Any of the requests which fail will not be included here, though an error will not necessarily be returned in the status.

bool px::cds::CCreateChannel::use_given_ids

If true, the channels will be created using the ID values given, as long as that ID is not already in use for any device or channel.

If false (the default), the system will assign IDs.
