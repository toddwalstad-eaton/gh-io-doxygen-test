Detailed Description

Represents a single alarm trigger condition.

Combines information from Mercury AlarmConfiguration and AlarmTrigger.

Member Data Documentation

uint32_t px::CAlarmTrigger::level

Trigger level.

0 is invalid. Latching alarms must be acknowledged at the highest tripped trigger level before they will close. For upper and lower bound, a higher-level trigger must be farther from 0 than any lower-level trigger. For px::alarm_trigger_type::in_between alarms, a higher-level trigger must be closer to 0 than any lower-level trigger.

CValue px::CAlarmTrigger::threshold

Threshold.

On sets, will be cast to the channel's px::CChannelSettings::value_type.

CValue px::CAlarmTrigger::threshold2

Second threshold, only used when when the trigger type is px::alarm_trigger_type::in_between.

bool px::CAlarmTrigger::latching

Latching alarm.

If true, stays open until the condition is cleared (channel value does not trip any triggers) AND the alarm is acknowledged at the highest level tripped. If false, once the condition clears the alarm is closed.

uint32_t px::CAlarmTrigger::delay

Delay in seconds that condition must hold true before triggering.

Unsigned so as not to disrupt space/time continuum. Currently ignored.
