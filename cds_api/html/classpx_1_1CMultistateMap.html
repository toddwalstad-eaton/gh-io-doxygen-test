Detailed Description

Maps variadically-typed keys into string label values.

This is most often used for boolean values when we want labels other than "true" and "false", or with integral values such as status codes, where each number has a specific meaning.

The type maps variably-typed values into corresponding string labels (an ordered multistate_map).

Was originally unordered_map, but changed to ordered map to get more consistent listing and output.
