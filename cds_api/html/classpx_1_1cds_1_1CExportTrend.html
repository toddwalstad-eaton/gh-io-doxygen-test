Detailed Description

Exports data from the trend storage to a CSV file.

After the file is written, the CDS will respond to this message, setting "success" to true if the file was exported successfully or to false if not. If "success" is false then "error" may be set to a string describing the problem.

Permissions Required:
px::permissions::data_read

std::vector<uint32_t> px::cds::CExportTrend::channel_ids

A list of channel ids in the desired order to export trends from.

If empty, no trends are exported.

unsigned int px::cds::CExportTrend::period

Indicates the period to export (aggregation will be applied if the stored data's period is smaller than the requested period.

This parameter is ignored if only export from stage or bucket, but not both.
