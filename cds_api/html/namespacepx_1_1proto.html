enum px::proto::ChangePasswordStatus : uint8_t
strong

Result status from password-change operations.

Enumerator
SUCCESS 

Password successfully changed.

PERMISSION_DENIED 

Caller lacks permission to change this user's password.

NONEXISTENT_USER 

The user does not exist.

SAME_PASSWORD 

The new password is the same as the previous password.

Expiration times are not updated in this case.

TOO_SOON 

Password policy specifies a minimum time between password changes, and it has not yet passed.

TOO_SIMPLE 

Password does not meet policy requirements, such as mixed case or minimum length.

EXISTING_PASSWORD_INCORRECT 

The operation required the existing password (currently password change operations do not require it), and it was incorrect.
