Macro Definition Documentation

#define TOKEN_ALLOC_SIZE   token_array_size

For PX Red, we will use a dynamically sized array (as long as the compiler permits), where the variable named 'token_array_size' must be given the desired size, eg with 'unsigned int token_array_size = iot_getJsonTokenCount( msg, len );'.

Declare the fixed allocation size of the tokens[] arrays we will use when deserializing JSON.

For PX Green, we fix this at a reasonably large size of 40, to handle the UpdateFirmware command.
