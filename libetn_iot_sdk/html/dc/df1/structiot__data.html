Detailed Description

Structure containing the data to be published for one message type (i.e.

DeviceTree, DeviceRealtime, ChannelRealtime, Trend etc) from one device. The values or data are stored in an Azure list of items. This structure is passed as a parameter to the function "iot_send".

unsigned long iot_data::messageId

Unique Id assigned to the message by iot_send().

This field is set during the iot_send() operation and is available upon return. It can be used by the application code to track the success (acknowledgement) or failure of the message.
