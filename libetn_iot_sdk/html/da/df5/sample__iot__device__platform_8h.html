void platform_message_thread_sleep ( )

Do a platform-specific sleep to pace the message thread.

When the thread resumes, it will check for new messages received. Commonly sleeps for 100ms or 1s. The Azure specs say that the platform must implement ThreadAPI_Sleep(), whether or not they do threading, so that function is to be used here as well.
