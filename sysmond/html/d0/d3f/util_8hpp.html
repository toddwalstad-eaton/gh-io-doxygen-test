std::vector<px::string> get_rwx_mounts ( const px::string &  mounts_file)

Return list of read/write executable mount points.

These are all filesystems that are mounted read/write without "noexec", except for virtual filesystems that root cannot make files in, like sysfs/proc. AUFS and Overlay mountpoints are also excluded (executability is determined by branch FSes).

Parameters
mounts_fileFile to read from (usually "/proc/mounts", parameterized for unit testability)
Returns
