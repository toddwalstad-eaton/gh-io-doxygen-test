Detailed Description

Class to configure, monitor, and update one system channel.

Given the sysmond.conf instructions for this channel, creates the channel if it doesn't exist, translates the directions into a monitoring process, and updates the channel when directed to.

bool MonitoredChannel::initialize ( const std::string &  channel_name,
const CONF_PAIR_ARRAY_T &  pair_array 
)

Given the channel name and its monitoring parameters, translate these into monitoring directions.

Create the channel under the publisher device, if it doesn't exist. If this initialization fails, this instance should be discarded and not called again for update_channel().

Parameters
channel_nameThe MCL name for the channel that will be created and updated to show the results of the monitoring directions.
pair_arrayKey-value pairs of monitoring directions from sysmond.conf.
Returns
False if the directions cannot be understood or followed, or if the channel cannot be created. True in the (normal) case of success.
