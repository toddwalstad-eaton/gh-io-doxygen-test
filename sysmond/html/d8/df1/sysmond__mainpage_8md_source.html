13 ## System Monitor
14 
15 This process serves to monitor the PX Red system for
16 
17 * System Integrity
18 * Channels of "Metadata" information
19 
20 This process was formerly known as "integrityd" when it only performed
21 the system integrity monitoring function.

47 # 3. Features {#mainpage_features}
48 
49 * Integrity monitoring
50  * Periodically checks integrity of the software by verifying that the squashfs backing file for the read-only filesystem has not
51  been replaced or tampered with, and that no executables have been put into read-write overlays.
52  * Reports status into channel aSoftwareIntegrityCheckFailed on device 1, creating it if it does not exist.
53  * Defines command cIntegrityScanNow on device 1; if received, runs an immediate scan instead of waiting for the next scheduled one.
54 * Metadata Channel monitoring
