std::vector<uint32_t> SysmondCore::commands_in_progress
private

List of handles of commands currently being processed.

Usually this will be empty or have one entry. But if a command comes in while a scan is in progress, it will be added to this list and all outstanding will be updated as complete (or failed) when the single scan is done.
