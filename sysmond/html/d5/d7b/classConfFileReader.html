Detailed Description

Class to read and process the contents of a .conf file.

Features:

  • Skips comments and blank lines
  • Handles one level of hierarchy
  • Provides map of contents

Handles .conf file contents such as

Some comment

Ipv4Address = { oob_tag = NETWORK_INTERFACE/1/network_interface field = ipv4_address hidden true }

Next is some variable with no depth

SimpleVariable = default_method

Similar with no equal sign

AnotherVar another_value

This would return a map of "Ipv4Address": array of 3 pairs "oob_tagoob_tag", "NETWORK_INTERFACE/1/network_interface" "field", "ipv4_address" "hidden", "true" "SimpleVariable": array of 1 pairs "default_method", "" "AnotherVar": array of 1 pairs "another_value", ""

std::pair< std::string, std::string > ConfFileReader::getKeyAndValue ( const std::string &  line)
staticprivate

This function attempts to return a pair of key and value tokens from the given configuration string.

They are assumed to be separated by either a space or an equal sign; all contents after the first separator are returned as the second string in the pair. If there is only one token, such as a closing brace, that is returned in the first string. Should there be no contents, a pair of empty strings will be returned.

Parameters
lineConfiguration line to process
Returns
