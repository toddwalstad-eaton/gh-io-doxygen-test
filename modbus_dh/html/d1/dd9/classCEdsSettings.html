Detailed Description

Class to manage Device EDS files.

This class will get the map of Device Model to EDS Filename, transform it into something that is useful in the Device Handler, and answer queries from the DH about EDSs.

It also creates and manages the map of parsed EDS data, attempting to minimize memory allocation by freeing parsed EDS data once it has been used by all interested Devices.

void CEdsSettings::garbage_collect_eds_map ( px_oob::PortProtocolTypes  protocol)

Use this function to periodically check to see if we can free up some parsed EDS data.

For the given protocol's Parsed EDS map, for each entry, check if we are holding a unique reference to the parsed data, meaning that all interested devices have finished with it, and it thus can be freed.

Parameters
protocolThe protocol (eg, Modbus) whose map should be garbage collected.
