CCommand::CCommand ( uint64_t  id,
px::CValue &  value,
px::datatype  cds_type = px::datatype::invalid 
)

Constructor to be used when provisionally creating this Command Parameter from EDS info.

A transient ID will be assigned to this Command Parameter by the DH. Normally the value will be set to 0 with the Command Parameter's type. Later, if the Command Parameter doesn't exist (fails its constraints test), then this Command Parameter will be removed.

Parameters
idThe unique ID for this parameter.
valueA starting value for the Command Parameter, of the right type. May typically be 0 or empty string.
cds_typeThe desired px::datatype for this parameter. This value typically comes from the device EDS file.
