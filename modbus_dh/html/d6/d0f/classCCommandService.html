Detailed Description

Command Service.

This class provides a testable, easy-to-use interface for finding and manipulating command-related information in the CDS.

Send a command result to a device in the CDS (overloaded).

Note: This overload performs a limited CDS update of CDeviceCommand members. In particular, the CDeviceCommand parameters are not updated.

Note: This overload can be used in cases where the CDeviceCommand param is const and the device_command_status needs to be updated.
