Check if a IPv4 address responds to ICMP ping requests.

Uses raw socket functions to perform ICMP ping instead of Boost ASIO socket functions.

Parameters
targetThe target IPv4 address.
timeoutHow long to wait for a ping response. In milliseconds.
Returns
Returns 1 if ping succeeds. 0 if ping fails (times out), -1 on internal error.
