typedef struct mbtcp_hdr_s mbtcp_hdr_t

Define the Modbus/TCP Header as a structure that our tests can cast from an array of uchars to get the fields by name.

Use reinterpret_cast<mbtcp_hdr_t*>( uchar_array )

There is an endian issue: Modbus is Big Endian, and x86 is Little Endian. So you have to use ntohs() with these 16-bit fields when used in our tests.
