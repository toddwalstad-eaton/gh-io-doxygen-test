Detailed Description

Channel class.

This is an externally-exposed channel. The primary difference between a channel and an internal parameter is that channels are known to the rest of the Toolkit. The DH treats internal parameters and channels more or less the same in its steady state (with the exception that channels have their values pushed to the CDS, of course), but channels are the only parameters which are visible to other parts of the system.

This class is largely a container for the fields that need to be pushed to the CDS to update one channel, using CUpdateChannelValue.

If the px::CChannelSettings are desired, they can be fetched using the Channel's ID with CGetChannelSettings. Since they are not needed on an on-going basis, they are not retained by this container.
