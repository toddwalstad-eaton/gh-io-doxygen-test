enum CDeviceSpecificHandler::WaveformStateEnums

The set of possible waveform acquisition states in an object's state machine.

The normal progression of the state machine is as follows: WAVEFORM_INITIALIZING - this is the initial startup state where the object is initializing and preparing to commence waveform acquisition.

0) WAVEFORM_INITIALIZING - the object is initializing and is ready to commence waveform acquisition. See the incom device handlers for device specific states.

When state is set to WAVEFORM_HALTED, the state machine will remain in the halted state (i.e. shutdown).
