Detailed Description

Features of Modbus/RTU device support

  • Connected on RS-485 (or RS-232) serial lines
  • User-configurable baud rates
  • For Modbus TCP, each device’s data appears twice:
    • Once as CDS “cached” data
      • mapped by us to Modbus register space
      • with device addresses we assign
    • Once as a pass-through device
      • using the device's native Modbus register map
      • retaining the physical device address
  • Modbus is a loosey-goosey “standard”, so extensive EDS specification for data transforms is necessary
