Detailed Description

Define the data provided by the EDS for one Channel or Internal Parameter.

The EDS tells us whether this is an Internal Parameter ("Variables") or Channel.

px::string eds_param_entry_s::canonical_name

The MCL/Custom name of this Channel or Internal Parameter.

If this Channel is a a duplicate of other channels in the EDS, then this string will be a custom name consisting of the canonical name plus a unique identifier appended to the end of the name. The alt_canonical_name must also be defined in the duplicate channel case.
