Detailed Description

CSerialPort class.

This is the serial-port-specific derivation of CPort, which adds the special functionality for Modbus Serial control.

bool CSerialPort::initialize_devices ( const std::shared_ptr< CAsioCdsHandle > &  cds_asio_handler,
const std::shared_ptr< CEdsSettings > &  eds_settings 
)

Initialize this port's devices and their configuration, and add these to the port's device list.

  • For all the Devices under this Port, using CDeviceService::find_child_devices()
    • gets their device information from the CDS
    • reads their persistent Out-of-Band (OOB) device configuration
  • For each Device
    • Parse the device EDS (if not already done)
    • (Try to) Add the new CDevice instance to the Port's device map.

The Devices must have been already provided to the CDS, and configured (ie, we don't do device discovery).

Parameters
cds_asio_handlerShared pointer to the CDS Handler from the ASIO framework.
eds_settingsShared pointer to the class that manages EDS files.
Returns
True if any devices were found and configured and enabled.
