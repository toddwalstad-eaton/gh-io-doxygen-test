Detailed Description

Base "parameter" class for Channels and Internal Parameters.

This class ties together some common functionality between Internal Parameters and Channels, so that the Device Handler can treat them the same for most operations.

Internal Parameters and Channels have 3 forms: 1) Simplest case: fixed value (eg, from EDS) - no updates 2) Update directly from an xBlock's Data Block Translation (DBT) array value 3) Update via an equation

enum CParam::PARAM_STATES

Define the set of states that a Parameter can take.

Enumerator
Provisional 

This Param needs further evaluation (eg, Equation terms or Conditional dependency).

Pending_Update 

This Param is awaiting a first Value update to become Valid.

Valid 

This Param is Valid and has a Value.

Statically_Valid 

This Param is Valid and has a Static Value (ie, no DBT update will occur).

Disabled 

This Param has been Disabled but is not being Removed.

bool CParam::initialize_translation ( CDevice &  parent,
const eds_param_entry_t &  eds_descr 
)
protected

Initialize the equation or direct xBlock transform for this parameter.

Will also use a static value, if given by the EDS.

Effects: will set this parameter's state to Valid if we had a static value from the EDS, and will set to provisional if it has a valid dependency equation

We allow the following types of direct transforms from DBTs, and set our dbt fields (_dbt_xblock_id not shown; 0 is also the "don't care" default) as follows: array_index bit_position count xBlock5[2] (reference location) 2 0 0 (number of bytes accessed depends on our type, not specified) xBlock4[3][7] (bit reference) 3 7 1 (single bit) xBlock4[2][4..7] (bit range) 2 4 4 (4 bits in bit range, masked and down-shifted) xBlock7[8..15] (blob or string type) 8 0 8 (byte range, for binary blob and string types) xBlock7[8] (blob or string type) 8 0 1 (one byte/char, for binary blob and string types)

Parameters
parentA reference to this parameter's parent CDevice, so we can look up xBlocks and equation sources.
eds_descrThe EDS description for this parameter, from which we get and use translation: The equation (eg, Y=avg(mACVAB,mACVBC,mACVCA) ) or direct transform (eg, xBlock7[5]). dependency: optional conditional equation that indicates when param is valid, e.g. if (xModel == 1); static_value: Used if there is no translation provided but this is; in this case, the parameter is immediately valid command: Will be checked for some exceptions, such as EDS_EXT_CMD_SPRINTF
Exceptions
eds_parse_errThrown if there is an equation and it contains a parse error, or if there is an error in the DBT translation specification.
Returns
True if a translation or equation was found and parsed, or a static value, of if it just had a valid conditional dependency equation (eg, some Commands); else False if no way found to assign a value
