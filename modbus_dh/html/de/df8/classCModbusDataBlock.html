Detailed Description

Implementation of a "data block" for the Modbus protocol.

A Modbus "data block" corresponds to an address range, where the addresses represent either registers (input or holding), discrete inputs, or coils.

This class supports Modbus-serial as well as Modbus-TCP, and allows reads and writes to/from block data using messages with appropriate function codes.

err_t  error 
)
virtual

Callback invoked when this DataBlock's messages have been completed.

For the case of no errors, updates the timer and converts and stores base data values. For the cases of errors which are irrecoverable and fatal to the block, bumps the IO error counter and flags this block as invalid if too many consecutive errors are seen. After handling the error cases, invokes the Device callback to inform it that this block is done.
