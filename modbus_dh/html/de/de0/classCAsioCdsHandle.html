Detailed Description

ASIO CDS Handler class for the Device Handler.

This class extends px::asio::CCdsHandler to support the asynchronous submit() function, and provides the handle_response() implementation that will be invoked when the CDS has a response to the submit()'d requests.

This class also maintains the idle message map of requests which have been submitted and are awaiting CDS responses.

void CAsioCdsHandle::handle_response ( std::shared_ptr< px::cds::CRequestResponse >  resp)

Customized CDS response callback function for responses to (asynchronous) submit() calls.

This is invoked by the I/O service when a response is sent by the CDS; it is actually the CCdsConnection Actor that passes the responsibility for processing the response to this function.

This function prototype matches the CCallback::cds_response_callback() type.

Parameters
respPointer to the instance of the Request-Response-derived object that was returned here.
