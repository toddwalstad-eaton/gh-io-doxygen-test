Detailed Description

Encapsulating class for various utility functions.

This class encapsulates several functions that aren't tied to any particular part of the CDS architecture, mostly OS-level functionality.

const char * CUtil::_result_stringtable
staticprivate

String table for Device Handler error (result) codes.

Column[0] are the "short" strings which are basically the same string as the err_t enumeration (useful for debug, dev). Column[1] are the "long" strings which are intended to be informational & user-facing.
