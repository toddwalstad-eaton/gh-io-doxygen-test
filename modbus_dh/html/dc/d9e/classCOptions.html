Detailed Description

Configuration options class.

This class handles parsing & storage of configuration options for the server, typically set via the command-line.

unsigned int COptions::polling_algorithm

Selects the polling algorithm.

0 selects 'prioritzed' polling (default device_handler polling). [1,...,60] sets the time interval (in seconds) for polling of the highest priority block. 61 (future) i.e. 'jog' polling. For now, selects the default polling algorithm.
