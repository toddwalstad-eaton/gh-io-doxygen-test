Detailed Description

Simple Modbus Message class (for Data Block polling)

This class implements an I/O message for the relatively simple case of a Modbus request/response. Note that, strictly speaking, this could be either Modbus-serial OR Modbus-TCP (if we were using a Modbus-TCP speaking device as a slave during testing, for example). But the practical case would be a Modbus-serial message.

Since Modbus matches the "simple message" model, it doesn't really need to add any message functionality. This is only here as a placeholder; otherwise, the CMessageHandler could just create a CSimpleMessage directly.
