Detailed Description

See the LibPX documents for detailed information about the PX Toolkit's event-driven application framework using Boost ASIO (ASIO = Async IO library).

The ASIO Actors for the Device Handler are:

  • Serial Ports, to manage send (Tx) and receive (Rx)
  • Watchdog Timer
  • Signal Handler (Break, Kill, etc)
  • Messages from the asynchronous CDS functions, or notifications
  • Listening sockets for Pass-through requests
  • Anything else that requires IO

Note that nothing limits “Device Port” Actors to serial ports

  • Ie, could be socket with network connection to other devices for aggregation or simulation
  • ASIO makes this extension easy

See file sample_io_sequence.h for an illustration of ASIO activity flow for one message.
