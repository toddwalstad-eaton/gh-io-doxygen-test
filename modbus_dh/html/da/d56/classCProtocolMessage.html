Detailed Description

Base class for a non-cloned (singleton) protocol-based IO message.

This class implements a request and response message exchange; the request is put in the _wbuf and response in the _rbuf.

This message progresses through the io_message_state series of CIoMessage.

This message type is normally a singleton but could be used as part of a transaction series of related messages.
