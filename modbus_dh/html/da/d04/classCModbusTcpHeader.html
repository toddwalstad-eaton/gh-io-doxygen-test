Detailed Description

Modbus-TCP header.

This class implements the "Modbus Application Protocol Header" (MBAP header) found in section 3 of the "Modbus Messaging on TCP/IP Implementation Guide".

px::asio::parse_result_t CModbusTcpHeader::parse ( px::asio::iobuf_const_iter_t  start,
px::asio::iobuf_const_iter_t  end 
)
virtual

Try to parse the given buffer.

This call parses an input buffer, assigning internal parameters to match the parsed request wherever possible. If parsing is successful, these internal parameters are used to service the request.

get_parse_err() can be used to see any error code that is the result of parsing; this will be:

  • NO_ERROR on complete success,
  • UNDETERMINED_RESULT if more bytes need to be read first
  • DATA_VALIDITY_ERROR will flag the special case where the Protocol version is non-zero and this request should be dropped.
Parameters
startThe beginning of the input buffer
endThe end of the input buffer
Returns
The pass/fail result of parsing, along with the ending position within the buffer.
