Detailed Description

The purpose of this class is to create and update a number of "synthetic" channels of real-timish data.

The data will appear like real data, with a slowly time-varying cycle and per-sample random component on top of a nominal value.

The channels will be chosen to represent the various types of normal data: floats, negative numbers, accumulating values, and power factor.
