5 | File name | Description |

9 | `https.conf` | This is a sample file helps to define configuration information about a standard service. In this case this service will get its server certificate signed by the pki1 service. |

12 | `pki1_cn.sh` | This is a sample script which explains how a common name can be generated dynamically for a service |
13 | `callback-pki1.sh` | This is a sample file that describes how to handle different system operations, when a operation happens in PKI1 service certificates |
14 | `callback-https.sh` | This is a sample file that describes how to handle different system operation, when a operation happens in a HTTPS service certificate |
15 | `restart_https.sh` | This is a sample script used to restart all https related px-toolkit system services when a HTTPS certificate gets generated/updated |
