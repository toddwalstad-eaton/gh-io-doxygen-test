bool CMercurySetTrendConfigurationRequest::post ( response_ptr_t  resp)
virtual

The post() method has two states (if successful):

  1. UPDATE_CHANNEL_SETTINGS: Processes the list of ChannelSettings from the CDS, and builds up a CDS request to UpdateChannelSettings
  2. POST: Processes the list of (simple) CDS responses to UpdateChannelSettings and notes any failures.

Parameters
respThe CDS response from our callback.
Returns
True if done, else False if more to go (eg, before POST is done).
