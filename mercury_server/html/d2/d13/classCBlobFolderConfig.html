Detailed Description

Class to hold one entry in our map of folders for Binary Blobs (user-managed files).

The initial data maps from one FolderConfigurationMapping entry, then extends it to list child folders.

px::CBitset<px::permissions> CBlobFolderConfig::_write_privileges

Privileges required to write to this folder and its child folders.

These privileges will be automatically inherited by any child folders. If no privileges are included here, the folder is read-only (at least for the purposes of the BinaryBlob commands).
