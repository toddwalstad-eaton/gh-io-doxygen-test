Serialize the TCP Header and binary Protobuf message into the buffer.

The header version, API ID, UUID, and other header fields must be already set before serialize() is called.

Parameters
buf[out]The TCP Header and binary Protobuf message will be serialized into (appended to) this buffer. Serialize the TCP Header and binary Protobuf message into the buffer with given api specification.

***** Important: ***** The given msg_api is used in the header "_message_api" field for this call only! Subsequent calls will continue use the original msg_api in header field. The header version, API ID, UUID, and other header fields must be already set before serialize() is called.
