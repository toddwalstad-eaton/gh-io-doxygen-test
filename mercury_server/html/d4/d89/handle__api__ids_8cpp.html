const px::permissions __api_perm_map[px::proto::ApiId_ARRAYSIZE][NUM_PERMS]
static

Define the private table that maps APIs to the Permissions that they require.

Permit up to three Permissions to enable a given API.

The APIs are implicit in this table, corresponding to the table entry index. These APIs are defined in both px::proto::ApiId and px::mercury_msg_api_t enum sets, but these have been constructed to match one-to-one and start with an "Undefined" entry at 0.

The Permissions are as defined for px::permissions, but these correspond one-to-one with px::proto::Permission enums. Again, _zed is a convenient Unknown placeholder.
