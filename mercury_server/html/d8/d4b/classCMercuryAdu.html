void CMercuryAdu::serialize ( px::asio::io_buf_t &  buf,
px::mercury_msg_api_t  msg_api 
)
virtual

Serialize the Mercury Header and binary Protobuf message into the buffer with given Mercury API ID specification.

***** Important: ***** The given msg_api is used in the header "_message_api" field for this call only! Subsequent calls will continue use the original msg_api in header field.

The header version, UUID, and other header fields must be already set before serialize() is called.

Parameters
buf[out]The Mercury Header and binary Protobuf message will be serialized into (appended to) this buffer.
msg_apiThe mercury message API ID which identifies the message purpose/category.
