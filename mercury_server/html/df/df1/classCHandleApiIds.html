Detailed Description

A utility class to help manage the use of the Mercury Message API IDs.

Does the Permission lookup and checking for the Mercury Message APIs. Also has utility functions to convert between Request message types and the API IDs, and to create a Message Handler for a given API ID.

The implementation of this class also hides the tables that provide the mapping of each API request to its allowed Permissions and to its Protobuf mappings.
