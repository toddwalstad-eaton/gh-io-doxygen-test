Detailed Description

Implementation of Mercury "CMercuryDisableDeviceRequest" message.

Note
The DeviceSelector has a recursive flag, but recursion is inherent when setting device state, so this flag is redundant. With one caveat: if the Device of a specified ID is already at the new state, its child devices will not be altered.
