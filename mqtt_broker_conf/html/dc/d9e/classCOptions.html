Detailed Description

Configuration options class.

This class handles parsing & storage of configuration options set via the command-line.

bool COptions::show_disabled_items

Option to print Disabled items into the output conf file output.

Normally they are not included in the output; if enabled, they will be printed with the '#' to comment them out and keep them disabled from the MQTT Broker's point of view.

Definition at line 85 of file options.hpp.

bool COptions::show_item_details

Option to print the full item details into the output conf file output.

Normally just the terse "tool_tip" line is included in the output; if enabled, the Description line(s) will be printed instead, each line prefaced with the '#' character.
