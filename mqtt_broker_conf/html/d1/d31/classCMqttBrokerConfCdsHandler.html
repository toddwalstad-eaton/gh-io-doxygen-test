void CMqttBrokerConfCdsHandler::handle_response ( std::shared_ptr< px::cds::CRequestResponse >  resp)
virtual

Called by the ASIO I/O service when one or more (normally expected) responses are sent from the CDS.

We have to provide an implementation of this function to use the virtual CDS Handler class, even though we have no actual use for it (all of our CDS calls will be synchronous).

Parameters
respThe response from the CDS (wrapped in a shared pointer). The resp type is aliased as cds::response_ptr_t

display_map_t CMqttBrokerConfCdsHandler::get_mqtt_display_map ( bool  want_disabled_items)

Reads the MqttBrokerSettings OOB object and puts it into a map of GenericTableEntry entries.

The key for the map will be the display "rank" field, so we can later output the fields in order.

Parameters
want_disabled_itemsIf enabled, the DISABLED entries will be added to the map. Normally they are not included in the returned map.
Returns
The display map for the GenericTableEntry entries. Will be empty only if failed to access the MqttBrokerSettings OOB object.
