1 # snmpd_conf program
2 
3 The snmpd_conf program is Eaton code to transform the input from the SnmpDataSettings, SnmpMibs, and
4 SnmpUserSettings OOBs
5 (Out Of Band) objects into an snmpd.conf file to control the operation of the Net-SNMP snmpd daemon.
6 If any SNMPv3 users are configured (as encouraged now, in place of the insecure v1 community strings),
7 the snmpd_conf program also reads in their SnmpV3UsmUser/<username> OOBs and stores their (hashed) information
8 in the semi-hidden /flash/app/snmpd.conf file.
9 
10 The purpose of the SnmpDataSettings, SnmpUserSettings, and SnmpV3UsmUser/<username> OOBs are to define that input,
11 allowing the UI to change the
12 settings in those OOBs. The snmpd_conf program is run before starting the Net-SNMP snmpd daemon
13 (in [its initialization script](../snmp_mibs/rootfs/etc/init.d/snmpd)), and this program parses
14 those OOBs and uses the contents to generate the appropriate snmpd.conf output.
15 
16 The one rather unusual function of snmpd_conf is to arrange to capture and save the configuration for new
17 SNMP v3 Users; this is a four-step process:
18 1. The UI makes a new entry in the SnmpUserSettings OOB's "v3_users" section.
19 2. snmpd_conf is run to generate an snmpd.conf file with a partial entry for this v3 user.
20 3. snmpd is started; it processes the snmpd.conf file, and writes out an entry for this user
21 in the semi-hidden /flash/app/snmpd.conf file, with the passphrase hashed.
22 4. snmpd_conf is run again to capture the hashed entry contents and save them to SnmpV3UsmUser/<username>.
23 
24 For the complete description of all the snmpd.conf options, see the [snmpd.conf man page](http://www.net-snmp.org/docs/man/snmp.conf.html).
25 
26 Information about running the Net-SNMP snmpd daemon can be found on the [snmpd man page](http://www.net-snmp.org/docs/man/snmpd.html),
27 and the other programs and libraries in the Net-SNMP suite are also described near there.

30 ## snmpd_conf Usage
31 
32 The usage and command line arguments for snmpd_conf are:
33 
34 ```
35 Usage: snmpd_conf
36  [-D <verbosity>] [-Ns] [-h] [-hv] [-p <dir>] [-s] [-up <dir>]
37  [-v]
38 
39 -D : Set the application's debugging verbosity level
40 -Ns : Don't direct log messages to syslogd
41 -h : Print usage information to stderr and exit
42 -hv : Print detailed/verbose usage information to stderr and exit
43 -p : Directory path to which snmpd.conf is to be written.
44 -s : Update user OOB objects from config files
45 -up : Directory path to which snmpd.conf is to be updated with createUser.
46 -v : Print version information to stderr and exit
47 ```
48 
49 The arguments used in a particular target can be found with the `showcfg` argument
50 to the `/etc/init.d/snmpd` script:
51 
52 ```
53 snmpd_conf -p /etc -up /initlo/data/app
54 ```
