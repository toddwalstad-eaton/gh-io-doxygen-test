void CSnmpdConfCdsHandler::handle_response ( std::shared_ptr< px::cds::CRequestResponse >  resp)
virtual

Called by the ASIO I/O service when one or more (normally expected) responses are sent from the CDS.

We have to provide an implemention of this function to use the virtual CDS Handler class, even though we have no actual use for it (all of our CDS calls will be synchronous).

Parameters
respThe response from the CDS (wrapped in a shared pointer). The resp type is aliased as cds::response_ptr_t
