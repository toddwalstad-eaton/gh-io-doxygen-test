Detailed Description

Map of CDS devices and channels to OPC UA nodes.

Instantiate this class and use its services to

  • map new CDS Device/Channel on OPC UA Object/Variable Node
  • find CDS Device/Channel mapped to OPC UA Object/Variable Node
  • find OPC UA Object/Variable Node mapped to CDS Device/Channel.

All mapping is done on ID level. No object copy is done.

The map does not store the type of CDS object.
