std::string opcua::utils::debug::cds_tree_indent_str ( unsigned short  indent)

Create indent string for Device tree print.

The tree is constructed from ASCII characters, the indent specifies the depth of the record in the tree and impicates the spacing and connectors to parents and siblings.

e.g. Device | |Subdevice1 | |SubSbdevice |Subdevice2

the | and spaces are given by the indent parameter. Device has indent 0.

See Also
cds_device_tree_to_str
cds_channel_tree_to_str
Parameters
indentThe indent in the tree.
