DLLEXPORT void __assert_fail ( const char *  assertion,
const char *  file,
unsigned int  line,
const char *  function 
)

__assert_fail() is overridden when you link with libsyslog.

This means that any assertion failures will now direct their output through the CSysLog class, allowing assertion failures to be visible in the syslog.

You normally won't need to call this function directly, it's utilized by the assert() macro when NDEBUG isn't enabled.
