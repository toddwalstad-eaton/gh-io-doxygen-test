DLLEXPORT void pcd_register_handlers ( void  )

Register standard signal handlers.

While some signals are handled in an app-specific manner, others are pretty much always handled the same way. This call registers a signal handler to catch SIGSEGV, SIGBUS, SIGILL, and SIGFPE, which pretty much always indicate a bug in the application. The handler will output a message indicating which signal was caught (to stderr & syslog), then call pcd_trace().
