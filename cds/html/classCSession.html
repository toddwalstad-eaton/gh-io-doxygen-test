std::pair< px::string, CSession > CSession::CreateSubsession ( const px::string &  subsession_id,
const px::CBitset< px::permissions > &  perms_mask,
uint32_t  timeout,
uint32_t  hard_timeout 
)

Create a subsession to ourself, using some new parameters, and add it to our set of subsessions.

This subsession gets its permissions as those of the parent session less any set in the perms_mask. The timeouts are set to the lesser of the requested or the parent's timeouts. Will copy the other parameters from the parent.

Parameters
subsession_idThe subsession token (ID)
perms_maskMask, with bits sets for the permissions that this subsession does NOT want.
timeoutIdle timeout for this subsession.
hard_timeoutHard timeout (not refreshable) for this subsession.
Returns
A pair for the sessions map, with the session token for the subsession plus the new subsession.
