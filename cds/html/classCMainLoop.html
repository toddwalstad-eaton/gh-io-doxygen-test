Detailed Description

Main loop class.

This class is used to handle overall program set-up/shut-down, initialize semi-global objects, and perform the main processing loop.

Member Function Documentation

void CMainLoop::main ( )

Execute the main loop.

This performs the "real" (steady-state) work of the CDS. This call will block indefinitely, and only return when the CDS has encountered some sort of non-recoverable error, or has been shut down by a user signal.
