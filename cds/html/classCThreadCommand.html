Detailed Description

Base thread command class.

This ABC defines what constitutes a thread "command", provides a common interface for handling all thread commands.

Member Function Documentation

virtual bool CThreadCommand::execute ( CThreadBase *  ptr)
pure virtual

Execute the given command.

This part of the CThreadCommand class isn't common across commands, so it is supplied by the subclass. Execution of a command may occur immediately if the command is relatively simple, or it may involve the execution of sub-operations.

Parameters
ptrPointer to a thread object which requested execution of this command
Exceptions
worker_quitThrown if the worker thread needs to exit (utilized by the "quit" command, mainly)
Returns
True if there is a response

std::vector<std::unique_ptr<COperation> > CThreadCommand::operations

A list of operations to be enqueued upon completion of this command.

This can be used to chain actions between threads by issuing thread commands which post-execute new operations, which issue more thread commands, etc.
