typedef boost::ptr_multiset<COperation> COpListBucket::op_set_t
private

Set of unique pointers to operations.

This differs from op_ptrset_t in that it owns the operations. I.e. this supplants the op_deque_t / op_ptrset_t combination that would otherwise be used, and instead maintains a fully sorted set of operations.
