Detailed Description

Command.

This class enumerates all of the commands for the CDS.

Objects of this type are intended for use in lieu of the raw enumeration values wherever possible. Despite being first-class objects, they occupy only 1 byte, and impose virtually no overhead.

PUBLIC const char* CAPICommand::str ( ) const

Getter for this canonical string associated with this command.

Enum to string conversions are supported via an inverse lookup table that maps to the same string entries used by the parse() function. The lookup table is built as a static member, i.e. initialized the first time any part of your program calls this function. After that, string conversions are extremely fast.

Returns
A pointer to the string value in an internal lookup table

PUBLIC void CAPICommand::parse ( const char *  str,
unsigned int  len 
)
private

Parse an input command string and assign the corresponding command enum.

This function utilizes a gperf-generated perfect hash table to perform an efficient lookup of the given canonical command string, and find its associated enumeration value. Comparisons are case- sensitive, and must match the corresponding canonical string exactly or they will be considered invalid. The result is assigned to _val.

Parameters
strThe canonical string name of the assigned command
lenLength of the string; if 0, length will be determined automatically by looking for a NUL terminator
Exceptions
generic_errThe provided string is unknown or invalid
