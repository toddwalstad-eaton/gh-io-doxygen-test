Detailed Description

Configuration options class.

This class handles parsing & storage of configuration options for the CDS, typically set via the command-line.

Member Data Documentation

size_t COptions::thread_max_exit_delay

Maximum amount of time to wait for a thread to exit after we send it a "quit" command.

This is related to the max queue size, as well as the amount of time that each command can take in the worst case, but it rarely comes into play aside from during stress testing.

Strength factor to use for hashing new passwords.
