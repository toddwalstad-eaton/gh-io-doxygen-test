#define __DBG_FUNC_VERBOSITY   11

Use a verbosity level of 11 for PUSH_FUNC, etc.

Higher makes it less likely that these will be seen, and this level is one notch above the 10 normally used by the application code, so you will only see these Push and Pop messages for libetnlicensing code when you really want to.
