Detailed Description

Email server settings.

These settings are read from the CDS, and control how the Email server operates. Settings are items mirrored in the cds OOB objects. As such only items with which you would expect the web user or other proc to control should be in here.

See Also
CEmailConfig for an explanation of settings vs. configuration
