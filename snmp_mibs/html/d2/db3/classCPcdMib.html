void CPcdMib::channel_settings_callback ( const px::asio::CCdsDevice &  container,
const std::vector< px::CChannelSettings > &  settings 
)

Called by the CdsHandler ChannelSettings callback to allow MIB-specific processing of the ChannelSettings data.

We register this callback with the CSnmpCdsHandler.

This function will call each interested MIB table handler and pass the settings along with the corresponding device's container, so the MIB table can do what it wants with the settings data. Function template: cds_ch_settings_callback_t& channel_settings_callback

Parameters
containerA read-only reference to the container of CDS data for that Device.
settingsThe array of ChannelSettings from the CDS; should be a complete set for this Device.
