Detailed Description

Container for data for one row of the pcdDigitalOutputsTable, providing a single digital output information described by this row.

A device may have multiple digital outputs (rows), each field (column) describes a unique property about the output, including the current value of the output. Will be instantiated when a Device's Digital Output is added to our table.

bool CPcdDigitalOutputsTableEntry::check_add_table_entry ( std::shared_ptr< CPcdMib >  pcd_mib_instance,
const px::CDevice &  px_device,
const std::vector< px::CChannelSettings > &  settings 
)
static

Factory function which checks to see if we need to add a pcdDigitalOutputsTable entry for the given Device.

If we already have an entry for this Device, does nothing. Else will process the possible pcdDigitalOutputsTable columns ("nodes") for this Device to see if we have any valid columns. If yes, will add this entry to the table.

Parameters
pcd_mib_instancePointer to the overall PCD MIB class, which holds valuable items like the CDS Handler, MIB Mapping EDS data, and this MIB Table.
px_deviceThe CDS Device data for a valid device; we will check its list of Channels.
settingsThe set of Channel Settings for this Device; we need it to validate some of the columns of this table.
Returns
True if an entry was added for this Device; False if it already existed, or we failed to get any valid columns for it.
