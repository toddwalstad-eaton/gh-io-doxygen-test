Detailed Description

Container for data for one row of the pwrMeterMinAvgMaxTable, providing the available measures (column) information for the Device described by this row.

May be instantiated when a Device is added to our table.

bool CPwrMeterMinAvgMaxTableEntry::check_add_table_entry ( std::shared_ptr< CPwrMeterMib >  pwr_meter_mib_instance,
const px::CDevice &  px_device 
)
static

Factory function which checks to see if we need to add table row entries (min, avg, max) for the given Device.

If we already have an entries for this Device, updates if necessary. Else will process the possible pwrMeterMinAvgMaxTable columns ("nodes") for each row to see if we have any valid columns. If yes, will add each such row entry to the table.

Parameters
pwr_meter_mib_instancePointer to the overall MIB class, which holds valuable items like the CDS Handler, MIB Mapping EDS data, and this MIB Table.
px_deviceThe CDS Device data for a valid device; we will check its list of Channels.
Returns
True if an entry was added or updated for this Device; False if it already existed, or we failed to get any valid columns for it.
