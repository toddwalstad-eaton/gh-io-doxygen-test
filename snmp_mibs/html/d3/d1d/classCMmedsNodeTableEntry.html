Detailed Description

Abstract base class for data for one row of a table of CMmedsNode.

Some of the tables that will inherit this class have a secondary index; for those, we will pack the primary and secondary indices into one uint64_t row key by shifting the primary by 8 bits and OR'ing in the 8-bit secondary index.

Each of the MIB Tables using this class will inherit some functionality and then provide their differentiated functionality.

uint64_t CMmedsNodeTableEntry::get_row_key ( )

A getter which provides the value to use as the key for this row.

In the simple case where there is just a Primary Index, this is that index value (the _dev_idx that this row was created with). In the case of a table with a secondary index (non-zero _sec_idx), the key will be the Primary Index shifted up by 8 bits and the 8-bit Secondary Index OR'd in.

Returns
The value to use as the key for this row, consisting of the Primary Index and, where applicable, the Secondary Index.
