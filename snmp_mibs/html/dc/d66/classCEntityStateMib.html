Detailed Description

Class which manages the entityState MIB variables.

Takes care of initializing and registering the MIB variables, and handling the CDS device state change callbacks for this MIB.

We are not creating a separate MIB submodule (DLL) for this MIB, since it shares so much with the entity MIB, including the CDS Handler data.
