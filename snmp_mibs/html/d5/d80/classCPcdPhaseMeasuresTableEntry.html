Detailed Description

Container for data for one row of the pcdPhaseMeasuresTable, providing the available measures (column) information for the Device described by this row.

May be instantiated when a Device is added to our table. The Primary Index is the Device ID for these table entries. The Secondary Index is the Phase Number (1 for A or AB, 2, or 3).

bool CPcdPhaseMeasuresTableEntry::check_add_table_entry ( std::shared_ptr< CPcdMib >  pcd_mib_instance,
const px::CDevice &  px_device 
)
static

Factory function which checks to see if we need to add any pcdPhaseMeasuresTable entries for the given Device, for each of 3 phases.

If we already have an entry for this Device and Phase, does nothing. Else will process the possible pcdPhaseMeasuresTable columns ("nodes") for this Device to see if we have any valid columns. If yes, will add the entry to the table.

Parameters
pcd_mib_instancePointer to the overall PCD MIB class, which holds valuable items like the CDS Handler, MIB Mapping EDS data, and this MIB Table.
px_deviceThe CDS Device data for a valid device; we will check its list of Channels.
Returns
True if an entry was added for this Device; False if it already existed, or we failed to get any valid columns for it.
