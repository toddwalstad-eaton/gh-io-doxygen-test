Detailed Description

Abstract Base Class for handling BACnet services by managing the communication to and from the CDS, as ASIO Actors.

: we will only need this class if we implement async CDS calls. We may have add the CBacnetMessage instance as well. Not currently used, no CBacnetServiceHandler services created

The derived classes will implement specific BACnet services, make asynchronous requests to the CDS, and handle the response callbacks.
