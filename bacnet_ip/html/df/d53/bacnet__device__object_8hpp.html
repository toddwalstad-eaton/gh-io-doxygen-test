typedef std::map<BACNET_OBJECT_TYPE, OBJ_MAP> OBJ_MAP_LIST

Define the map for the maps of supported Object types.

Keyed by Object Type, paired with the OBJ_MAP entries.

Can access objects by Type and ID as though this map is a 2-dimensional array, eg, BacStructViewObject *pView = (BacStructViewObject *) mObjMapList[OBJECT_STRUCTURED_VIEW][viewID].get();
