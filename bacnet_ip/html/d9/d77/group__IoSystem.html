The ASIO Actors for bacnet_ip are:

  • Protocol handlers to manage BACnet services
  • Watchdog Timer
  • Signal Handler (Break, Kill, etc)
  • Messages from the asynchronous CDS functions, or notifications
  • Anything else that requires IO

See file sample_service_sequence.h for an illustration of ASIO activity flow for one service call.
