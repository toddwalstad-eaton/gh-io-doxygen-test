virtual bool CMockConfigCore::config ( )
virtual

This function sets up the configuration for the main loop().

It:

  • Creates the Manager for the ASIO framework
  • Creates IO Actors for the Signal Handling and Watchdog, and add them to ASIO
  • Creates an Actor to manage the CDS and add it to ASIO
  • gets the settings for BACnet/IP from the OOB object

Split from CBacnetIpCore constructor to facilitate Unit Tests. Done as virtual so the test class can override it.

Returns
True if the configuration steps succeeded, else False if BACnet/IP is not enabled or other fatal problem was detected.
Exceptions
init_errIf there was a problem initializing the main loop object. This can happen for any number of reasons; more specific information is printed at lower levels, and this error is propagated back to indicate that the caller should exit.
