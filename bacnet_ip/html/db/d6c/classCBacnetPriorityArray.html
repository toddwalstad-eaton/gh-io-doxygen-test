Detailed Description

template<class T>
class CBacnetPriorityArray< T >

This class implements the methods to handle BACnet Priority Arrays and Relinquish Default fields.

template class T - base type of the priority array. For example, BinaryOutput objects would have a CBacnetPriorityArray with T = bool, AnalogOutput objects would have a CBacnetPriorityArray with T = float.
