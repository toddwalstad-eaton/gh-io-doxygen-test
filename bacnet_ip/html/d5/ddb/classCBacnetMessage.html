Detailed Description

This simple class holds basic information for one received BACnet message.

The contents allow tracking of source and data through the calls in and out of the bacnet-stack, and will facilitate asynchronous CDS calls, if used.

Not every received BACnet message will result in one of these Messages; must be of type PDU_TYPE_CONFIRMED_SERVICE_REQUEST.
