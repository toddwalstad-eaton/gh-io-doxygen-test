Detailed Description

Abstract Base Class for BACnet Objects.

Mostly serves as a container for the properties common to all BACnet objects, and provides common access for these common properties.

This abstract class is not instantiated directly, but always by inheritance.

int CBacnetObject::ObjectReadPropertyFromList ( BACNET_READ_PROPERTY_DATA *  rp_data,
const BacPropEntry_t *  propEntries 
)

Reads one property for this object type, if it is listed as supported.

This function is invoked by the derived classes before they attempt to handle the property on their own. This function will handle those properties

  • basic and common to all objects
  • with generic handling based on the BacPropEntry_t information
  • If the property is found, but its appTag = BAPTG_CUSTOM, the property will be provided by the derived class, so this function returns the negative of the index into propEntries.
  • Finally, if the requested property is not supported, this function returns -1 and sets an Error.
Parameters
rp_dataPointer to the BACnet_Read_Property_Data structure, which is packed with the information from the RP request.
propEntriesPoints to the list of Properties for the derived object class. The final entry must have propID = PROP_ALL.
Returns
The length of the apdu encoded, or 0 for null request, -1 for error, -2 for abort message, or the negative of the index into propEntries for a property which must be provided by the derived class.

bool CBacnetObject::get_cov_value_state ( BACNET_PROPERTY_VALUE *  value_list)
virtual

Get the present value for a COV notification for this object, and fill out the two entries in the given value list with its value and state.

Calling this function will also clear the _has_cov_update flag, to await another update. The default implementation of this virtual function will always return False and not provide a value or state; object classes the can provide COV updates should override this with a useful implementation.

Parameters
value_listPointer to the value structure to be filled out; may be in a linked list of values.
Returns
True if this object supports COV and had a value available.
