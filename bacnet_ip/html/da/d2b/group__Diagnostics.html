The Modules pages for BACnet/IP are (mostly) aligned with the logging subsystems defined in log.hpp.

Enumeration Type Documentation

enum svr_log_lvl_t

These values are used to define server "subsystems" whose verbosity can be varied individually, to get more specific information from the data server without overloading the user with output.

Note: when updating this array, be sure to update the corresponding array COptions::subsystem_strings.

For Doxygen, each subsystem also has a corresponding "Module" defined in bacnet_ip.hpp and linked to the most important or interesting aspects of the submodule.
