bool CBacnetMultiStateValue::encode_present_value ( const px::CValue &  cds_value,
BACNET_APPLICATION_DATA_VALUE &  bac_value 
)
virtual

Encodes the present-value for a Point object from its CDS value.

If the return value is True, then the value and its type will have been set in the "type" union and tag field, respectively (and confusingly).

For Multistates, must iterate through the state values until we find one that matches the CDS value. And note that the resulting values are 1 to N, where N is the number of states.

Parameters
cds_valueValue of this point, already read in from the CDS.
bac_valueThe structure where the results will be placed on successful output.
Returns
True if we were able to encode the present value, and so it can be used.
