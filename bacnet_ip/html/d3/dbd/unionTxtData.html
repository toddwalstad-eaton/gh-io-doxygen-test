Detailed Description

BACnet vendorId The BacPropEntry_t struct has a member as text hint for the Property's value or field name Text hint for fixed value's are preceded with "#" and for field name's it's preceded with a "$" sign as first character.

Based on the first character of this hint[], a decision is made to get either a field name for the value or convert a fixed value. Thus we need to concatenate two strings i.e "#" and property value during compile time vendor id property. For this purpose below union has been created. Note: This union can also be used for any other prefix
