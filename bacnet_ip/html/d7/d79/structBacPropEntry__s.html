Detailed Description

Structure for our arrays of Properties for each Object.

For each Property, indicate its type, requirement, and path to access it.

The reqmt field should be one of PROP_REQUIRED, PROP_OPTIONAL, or PROP_PROPRIETARY for answering some rpm requests.

An appTag = BAPTG_CUSTOM means that we must hand off to the Object's derived class to process this one specially.

The path has multiple meanings, depending on the first character:

  • '$' is the name of a field in the object's CDS instance (eg, "$location") that the derived class will look up for us. Follows the rules for field names in GetFieldByName().
  • '#' indicates that what follows is a constant value, not a path, whose type depends on the appTag (eg, "#1476" for MAX_APDU)
  • Else (if appTag == BAPTG_CUSTOM) is hint text for the derived class to use in evaluating the property's value.
