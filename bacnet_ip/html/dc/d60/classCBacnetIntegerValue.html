BacPropEntry_t* CBacnetIntegerValue::_integerValueUnitsExtProps
protected

The extended propert list for this Integer Value object By default this will point to global property list for Integer Value object.

But when a instance object has additional required/optional/proprietary properties, then that instance will create a dynamic property list for that specific instance and will copy all global properties there and then add additional properties specific for that instance. So this will always points to the correct property list for that instance.

const int* CBacnetIntegerValue::_integerValueOptionalProps
protected

Pointer to the list of Optional Properties for this instance of Integer Value.

By default this will point to normal Optional property list for Integer Value object. But when a instance object has additional optional properties, then that instance will point to a modified Optional property list for that specific instance. So this will always points to the correct property list for that instance. This is semi-redundant with _integerValueUnitsExtProps, but intended for a different target; this list of just the properties is for filling out PROP_PROPERTY_LIST.
